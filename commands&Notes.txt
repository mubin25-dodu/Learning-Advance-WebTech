npm install class-validator 
            class-transformer 
            @nestjs/platform-express 
            multer 

notes :
    •class-validator — provides decorators like @IsString(), @IsNotEmpty() for DTO validation 
    • class-transformer — transforms plain objects into class instances 
    • @nestjs/platform-express — provides FileInterceptor, FilesInterceptor for file uploads 
    • multer — handles multipart/form-data; provides diskStorage 
    • @types/multer, @types/express — TypeScript type definitions (dev only)
